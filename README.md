# Data Analysis, Machine Learning & Power BI Dashboard Project

## Overview
This project demonstrates a complete data analysis workflow, including:

- Data cleaning and preprocessing using Python
- Exploratory Data Analysis (EDA)
- Building and evaluating Machine Learning models
- Creating an interactive Power BI dashboard for insights and visualization

---

## Project Files

- **Project_cleaning.ipynb**  
  Contains data cleaning, preprocessing, and exploratory data analysis.

- **Project_ML.ipynb**  
  Contains machine learning model building, training, and evaluation.

- **Dashboard.pbix**  
  Interactive Power BI dashboard for data visualization and business insights.

---

## Tools & Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Power BI

---

## Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis
4. Feature Engineering
5. Machine Learning Model Training
6. Model Evaluation
7. Dashboard Creation in Power BI

---

## Results

The project provides:

- Cleaned and structured dataset
- Insights from exploratory analysis
- Predictive machine learning model
- Interactive dashboard for decision-making
